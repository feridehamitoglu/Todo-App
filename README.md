# React + Vite

## Teknolojiler

- **React** – UI için
- **Tailwind CSS** – Hızlı ve modern tasarım için
- **JavaScript** – Fonksiyonellik ve state yönetimi

---

## Özellikler

- Yeni görev ekleme
- Görev tamamlama (checkbox ile)
- Görev silme
- Responsive tasarım
- Modern ve temiz UI (Tailwind CSS ile)

---

Bu proje, **React** ve **Tailwind CSS** kullanılarak yapılmış basit bir **ToDo List (Yapılacaklar Listesi)** uygulamasıdır. Kullanıcılar yeni görev ekleyebilir, tamamlanan görevleri işaretleyebilir ve silebilirler
